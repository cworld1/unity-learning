## Local Development

Environment requirements:

- [Unity](https://unity.com/): 2023+

Clone the repository:

```shell
git clone https://github.com/cworld1/unity-learning.git
```

Open with Unity Hub:

- Go to `Projects`
- Click `Add`
- Choose the peoject folder
